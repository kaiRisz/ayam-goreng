Keterangan

Jadi ada table, td, tr, th. Berikut keterangannya:

  • Table : tag utama buat bikin table
  • td : tag buat bikin isian tabel
  • th : tag buat bikin header table
  • tr : tag buat bikin baris table
  • colspan : buat merge tabel kesamping
  • rowspan : buat merge tabel kebawah
